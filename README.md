# Genomic basis of granulovirus resistance in *Plodia interpunctella*

This repository contains data and analysis code associated with the manuscript:

**"Genomic structure, not life-history strategy, predicts granulovirus resistance in *Plodia interpunctella*"**

---

## Repository structure

### `R_code/`

R scripts for all analyses reported in the manuscript, in order of execution:

| Script | Description |
|--------|-------------|
| `01_life_history.R` | Life-history PCA, LD50 calculation, dose-response GLMs|
| `02_genomic_pca.R` | Genomic PCA, neighbour-joining tree, PC1 vs LD50 correlation |
| `03_manhattan_wza.R` | Sliding-window PCA-LD50 correlation, WZA, Manhattan plots, QQ plots. **Run on Savio HPC.** Outputs `figures.pdf` and five `.rds` files for local plotting. |
| `03b_manhattan_plots_local.R` | Local plotting script for Figure 5 and QQ plots. Requires `.rds` files transferred from Savio after running `03_manhattan_wza.R`. Run in RStudio. |
| `04_clustering.R` | K-means clustering of LD50 values |
| `05_ROH_figure.R` | Runs of homozygosity analysis and figure |

Scripts should be run in order. `01_life_history.R` must be run first as it generates `ld50_calculated.csv`, which is used by all subsequent scripts.

#### Notes on `03_manhattan_wza.R` and `03b_manhattan_plots_local.R`

`03_manhattan_wza.R` is designed to run on the Savio HPC cluster at UC Berkeley and requires the `wza_cache/` folder of pre-computed per-window SNP files (generated by `parse_vcf_final.R`, not included in this repository due to file size). After running on Savio, transfer the five output `.rds` files to your local machine:

```bash
scp 'pswhite@hpc.brc.berkeley.edu:/global/scratch/users/pswhite/analysis/*.rds' ~/path/to/local/
```

Then run `03b_manhattan_plots_local.R` in RStudio to generate and tweak Figure 5. The FDR thresholds are hardcoded in `03b_manhattan_plots_local.R` based on 571 windows from the Savio run.

## Software requirements

All analyses were run in R. Key package versions:

- `MASS` — LD50 estimation via `dose.p`
- `vcfR` v1.16.0 — VCF file handling
- `adegenet` v2.1.11 — genomic PCA
- `ape` v5.8 — neighbour-joining tree
- `SNPRelate` v1.38 — sliding-window PCA
- `ggplot2`, `patchwork`, `ggrepel` — figures
- `gridExtra`, `grid` — table and layout rendering

Computationally intensive analyses (sliding-window PCA, WZA) were run on the Savio HPC cluster at UC Berkeley (`03_manhattan_wza.R`) under account `fc_sirmodel`.

## Genomic pipeline

Upstream variant calling was performed on the Savio HPC cluster. Per-sample sorted VCF files were compressed and indexed using `bgzip` and `bcftools` v1.16, merged using `bcftools merge`, and filtered to coding sequence variants using `bedtools intersect` v2.31.0 against the *P. interpunctella* reference genome annotation (GCF_027563975.2). Sliding windows of 1 Mb with 500 kb steps were generated using `bedtools makewindows`. Runs of homozygosity were identified using `bcftools roh` v1.16. The upstream assembly and variant calling pipeline was performed by J. Lo (University of California, Berkeley).
